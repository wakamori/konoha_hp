Downloads

KonohaScript is an open source software, and this page shows the way of downloading. KonohaScript is available for many platforms and you can download with preferable way that suit for your platform. We prepare binary packages including installers, source tarball, and svn repository.

SVN Repository

The latest version of KonohaScript is available from subversion. You can checkout the repository, configure, and build. The instructions below shows how to use svn repository.

$ svn checkout http://konoha.googlecode.com/svn/trunk/ $(konohascript) $ cd $(konohascript)/konoha $ cd build $ cmake .. $ make $ sudo make install
Development Base

The development base of KonohaScript is hosted by googlecode.com. The latest information is also available at development base including the information for developers.
