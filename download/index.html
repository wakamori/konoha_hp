SVN Repository

NOTE: To build Konoha, you shuold install CMake (v2.8+) The latest version of KonohaScript is available from subversion. You can checkout the repository, configure, and build. The instructions below shows how to use svn repository.

$ svn checkout http://konoha.googlecode.com/svn/trunk/ $(konohascript) $ cd $(konohascript)/konoha $ cd build $ cmake .. $ make $ sudo make install
